# Hegselmann-Krauze with leaders model of group dynamics

This is a group project completed for the course of Group Dynamics, studying a nontrivial modification of the Hegselmann-Krauze model of opinion dynamics.

## Project description

The objective of this project was to find a mathematically reasonable way of integrating a notion of "leaders" - people with a high tendency to influence other peoples opinion - into the well-known Hegselmann-Krauze model of opinion dynamics, establishing the correctness of the modified model, and running some simulations.
